# UML_Diagrams
Designed the UML Diagrams for a Library Management Systems OOD with Singleton and Factory
## Library Management System (C++)
 ### Design Patterns Used- Singleton → Only one Library instance- Factory → Creates Book and Member objects
 ### Features
 - Add Books
 - Add Members 
 - Borrow / Return Books
 - Show All Books
 ### Build (VS Code)
 Use: C/C++ Extension + MinGW
